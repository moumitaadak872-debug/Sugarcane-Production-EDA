## 📖 About the Project

This project performs Exploratory Data Analysis (EDA) on a Sugarcane Production dataset to uncover trends, patterns, and relationships among different 
agricultural variables.
The analysis involves data cleaning, preprocessing, statistical summaries, and interactive visualizations to better understand sugarcane cultivation and production.


## 🎯 Objectives

- Understand the structure of the dataset
- Clean and preprocess the data
- Analyze sugarcane production trends
- Compare production across different states
- Identify relationships between production, area, and yield
- Generate meaningful insights through visualization

---

## 📂 Dataset Information

The dataset includes agricultural information such as:
- State
- Year
- Area under cultivation
- Sugarcane Production
- Yield
- Other production-related attributes

File Format: CSV

🔎 Exploratory Data Analysis

The notebook investigates questions such as:

- How many records and features are present?
- Are there any missing values?
- What are the statistical summaries?
- Which states have the highest sugarcane production?
- How does cultivation area influence production?
- Is there any correlation among numerical features?
- Are there any outliers?
- What trends can be observed from the data?

▶️ Getting Started

Clone the repository

```bash
git clone https://github.com/your-username/Sugarcane-Analysis.git
```

Go to the project directory

```bash
cd Sugarcane-Analysis
```

Install dependencies

```bash
pip install pandas numpy matplotlib seaborn
```

Launch Jupyter Notebook

```bash
jupyter notebook
```

Open Sugarcane_Analysis.ipynb and run all the cells.
